# Дашборд мониторинга ключевых показателей авиакомпании

## Описание проекта

Проект посвящён анализу ключевых метрик авиакомпании с целью оценки эффективности бизнеса, структуры флота и ценовой политики.

Разработан интерактивный дашборд в **Apache Superset**, объединяющий данные о выручке, пассажирских сегментах и характеристиках воздушных судов.

---

## Бизнес-задача

Авиакомпания сталкивается с необходимостью:
- понимать динамику выручки во времени
- оценивать вклад различных классов обслуживания в доход
- анализировать эффективность разных моделей самолётов
- поддерживать принятие решений по оптимизации флота и ценовой политики

---

## Роль в проекте

Финансовый / продуктовый аналитик:
- разработка SQL-запросов
- построение витрин данных
- визуализация в Apache Superset
- формирование бизнес-инсайтов на основе данных

## Структура дашборда

### 1. Динамика выручки по дням и классам обслуживания
Анализ ежедневной выручки с разбиением по классам (Economy, Comfort, Business).

Позволяет:
- выявлять сезонность спроса
- оценивать вклад каждого класса в общий доход
- отслеживать изменения структуры выручки
<img width="1280" height="776" alt="image" src="https://github.com/user-attachments/assets/fc62ef51-5d65-4233-bce5-e27e4c2c34d3" />

```sql
select 
    fare_conditions, 
    DATE(actual_arrival) as date, 
    sum(amount) as day_sum
from ticket_flights 
join flights using(flight_id)
where status = 'Arrived'
group by fare_conditions, DATE(actual_arrival);
```

### 2. Распределение посадочных мест по моделям самолётов

Анализ конфигурации флота и структуры классов обслуживания в разрезе моделей ВС.

Позволяет:

оценить вместимость самолётов
сравнить структуру классов между моделями
выявить особенности эксплуатации флота
<img width="1270" height="771" alt="image" src="https://github.com/user-attachments/assets/3659dfd2-462d-4001-b507-d22a53697efa" />

```sql
select 
    model['ru'], 
    fare_conditions, 
    COUNT(*)
from seats 
join aircrafts_data using(aircraft_code)
group by 1, 2;
```
### 3. Средняя стоимость билета по моделям самолётов

Анализ ценовой политики компании в зависимости от используемого воздушного судна.

Позволяет:

сравнить доходность различных моделей самолётов
выявить наиболее прибыльные типы ВС
оценить различия в среднем чеке
<img width="1280" height="568" alt="image" src="https://github.com/user-attachments/assets/39ef896f-c97b-4e27-8160-0a182f9cd284" />

```sql
SELECT 
    model['ru'] AS model,
    AVG(amount) AS avg_price
FROM ticket_flights
JOIN flights USING(flight_id)
JOIN aircrafts_data USING(aircraft_code)
GROUP BY 1
ORDER BY avg_price DESC;
```

### 4. Уточнённый расчёт средней стоимости билета

Используется для более точной финансовой отчётности.
<img width="1280" height="553" alt="image" src="https://github.com/user-attachments/assets/e568a885-637b-4b2f-a158-22274c735090" />

```sql
select 
    model['ru'] as model, 
    ROUND(AVG(amount), 2) as avg_price
from ticket_flights 
join flights using(flight_id)
join aircrafts_data using(aircraft_code)
group by 1;
```

## Основные бизнес-инсайты
Бизнес-класс формирует наибольшую выручку на одного пассажира
Разные модели самолётов демонстрируют существенные различия в среднем чеке
Наблюдаются колебания выручки, что может указывать на сезонность спроса
Структура флота влияет на доходность рейсов и распределение классов обслуживания
